# NSH Shell

A C implementation of a minimal Bash Shell

## Features

* infinite command length
* O(n) command parsing
* Absence of Memory Leaks
* Infinite Piping
* File Redirection (Input and Output of stdout)
* Scripting
* Background execution of commands

## Compile

Compile with the given makefile

`make`

## Running

### Interactive Mode

`$ nsh`

### Script Mode

`$ nsh [script]`
